RS = Digital pin  1
RW = Arduino GND
E = Digital pin 2
D4 = Arduino digital pin 3
D5  = Arduino digital pin 4
D6 = Arduino digital pin 5
D7 = Arduino digital pin 6
A = Arduino  5V
K = Arduino GND

Ultrasonic trig =  Arduino digital pin 9
